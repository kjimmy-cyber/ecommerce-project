# 🛒 E-Commerce Project (Node.js + Prisma + MySQL)

A full-stack e-commerce web application built using **Node.js, Express, Prisma ORM, MySQL, HTML, CSS, and JavaScript**.

This project includes authentication, product management (CRUD), shopping cart system, and admin dashboard.

---

## 🚀 Features

### 👤 Authentication System
- User registration
- User login
- Password hashing using bcrypt
- JWT authentication
- Role-based access (Admin / User)

---

### 🛍 Product Management (CRUD)
- Add new products (Admin)
- View all products
- Update products
- Delete products
- Product description and price handling

---

### 🛒 Shopping Cart System
- Add products to cart
- Remove items from cart
- View cart page
- LocalStorage-based cart system
- Total price calculation

---

### 🧑‍💼 Admin Dashboard
- View all products in table format
- Edit products
- Delete products
- Simple admin control panel

---

## 🧱 Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **ORM:** Prisma  
- **Frontend:** HTML, CSS, JavaScript  
- **Authentication:** JWT + bcrypt  
- **Storage:** LocalStorage (cart system)
